#### 一. 插件基本介绍
http 内部封装axios js-cookies
1. 实现功能
```
   1. 全局通用配置环境、基础功能配置 参考 RequestDefaultConfig.js文件
   2. 单个请求配置
   3. token保存以及验证
   4. get,post,put,delete等请求的promise封装
   5. 取消重复请求
   6. 取消当前页面全部请求  
```

#### 二. 插件安装

>  npm install lzb-http

#### 三. 插件使用案例


1. 导入文件
```javascript
import  http from "lzb-http"
```
